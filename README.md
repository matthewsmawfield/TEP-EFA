## Abstract

Twelve Earth gravity assist flybys (NEAR, Galileo 1990/1992, Cassini, Rosetta 2005/2007/2009, MESSENGER, Juno, Stardust, OSIRIS-REx, BepiColombo) are analyzed using the Temporal Equivalence Principle (TEP) framework with a chameleon scalar field. This analysis tests the TEP framework's prediction: a universal conformal coupling A(φ) = exp(2β φ/M_Pl) between matter and a dynamical time field φ, with the coupling constant β constrained by observations. The TEP model predicts a scalar fifth force F = β_eff c² ∇φ/M_Pl arising from the conformal coupling of the scalar field φ to matter, where β_eff = β × (ΔR/R) includes thin-shell screening. The scalar field relaxes outside Earth with a screening length λ_TEP ≈ 4000 km, established independently from GNSS atomic clock correlations. The non-radial component of this force—modulated by Earth's oblateness (J2) and the spacecraft's trajectory asymmetry factor cos δ_in - cos δ_out—produces a net velocity change that appears as the flyby anomaly. The model predicts null results for symmetric trajectories (Galileo 1992, MESSENGER) and fits three primary detections (Rosetta 2005: 1.82 mm/s, Galileo 1990: 3.92 mm/s, NEAR: 13.46 mm/s) with fitted β values spanning a factor of 2.23 (8.87×10⁻⁵ to 1.98×10⁻⁴, weighted mean 1.29×10⁻⁴). This represents a significant improvement over the prior model's ~100× scatter, suggesting that trajectory asymmetry and the TEP screening length may be relevant physical ingredients. All fitted β values satisfy the Cassini PPN bound |γ - 1| < 2.3×10⁻⁵ with a safety margin exceeding 600×. Bayesian model comparison favors TEP (88% evidence weight) over null (0%) and empirical (12%) alternatives. The TEP scalar force model offers a self-consistent, PPN-compliant explanation for the Earth flyby anomaly that connects to dark energy physics through the universal critical density ρ_c ≈ 20 g/cm³ and screening radius R_sol ≈ 4200 km established from cross-paper analysis of the TEP framework.

## Key Findings

Three successful TEP fits to flyby anomalies (Rosetta 2005, Galileo 1990, NEAR 1998) with fitted β values spanning a factor of 2.23 (8.87×10⁻⁵ to 1.98×10⁻⁴, weighted mean 1.29×10⁻⁴). All fitted parameters satisfy solar system PPN constraints (|γ - 1| < 2.3×10⁻⁵) with a safety margin exceeding 600×. The model predicts null results for symmetric trajectories (Galileo 1992, MESSENGER) and explains most null results through trajectory geometry. Bootstrap (n=10,000) and leave-one-out cross-validation indicate stability.

## Installation

```bash
# Clone repository
git clone https://github.com/matthewsmawfield/TEP-3I.git
cd TEP-3I

# Install dependencies
pip install -r requirements.txt
```

## Reproduction Steps

The flyby TEP analysis pipeline is fully automated. Run the complete pipeline:

```bash
python scripts/flyby_pipeline/run_all.py
```

This will populate `results/` and `data/processed/` with fresh analysis outputs.
